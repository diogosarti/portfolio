# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/diogosarti/portfolio.git

# entrar na pasta do projeto front end web
cd portfolio

# instalar dependências
npm install

# executar o projeto
npm run dev
```
